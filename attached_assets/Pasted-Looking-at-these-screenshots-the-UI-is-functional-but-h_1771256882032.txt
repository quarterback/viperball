- **Team colors:** Pull from team data, use as accents
- **Scheme colors:** Consistent color per scheme type

  - Lateral Spread: Blue
  - Kick Heavy: Green
  - Ball Control: Gray
  - Balanced: Purple
- **Conference colors:** Each conference gets a subtle background tint

Current hierarchy is decent. Tighten up:

- **H1:** Page titles (Viperball Game Simulator) â€” keep as-is

- Add more vertical breathing room between team cards

## Streamlit-Specific Notes

Streamlit has limitations. Some suggestions for working within them:

**For team cards:** Use `st.container()` with custom CSS via `st.markdown()` with `unsafe_allow_html=True`

**For drag-and-drop:** Consider `streamlit-sortables` or similar community component

**For tooltips:** Use `st.help()` or `st.popover()` (Streamlit 1.29+)

**For color swatches:** Inline HTML/CSS or small colored `st.container()` backgrounds

**For radar charts:** `st.pyplot()` with matplotlib or `st.altair_chart()`

If Streamlit becomes too limiting, consider migrating to Gradio (more flexible layout) or a full React frontend with FastAPI backend.
